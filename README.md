**Project Title:**
In Depth Shutdown Commander

**Description:**
A console application with commands to execute system proccesses for power options such as sleep, hibernate, shutdown. Timer commands are included to delay the process based on user provided input.

**Excuting Program:**
* Simply start the application file and provide one of the accepted commands in order to execute a process.
* If you aren't sure what commands are available, simply type Commands (not case sensitive) and a list of commands will be displayed to the user.

**Version History:**
1.0
  * Initial release

**Help:**
Help commands and information are provided in the application by simply typing Help (not case sensitive).
